우클릭 > 클래스 > 상단 패키지명도 기재


*단축키  
  ctrl + F11 : 실행 
  ctrl + shift + 방향키 : 단어 블럭 처리
  alt + shift + 방향키 : 한줄 블럭 처리  

1. 제어문 : 선택문 + 반복문 + 기타 구문 

1) 선택문 : 프로그램의 흐름을 조절하거나 순서 변경을 수행하는 구문 
              If 구문 , switch case 구문 


2) 반복문 : 특정 구문을 반복적으로 수행하는 구문
	  for 구문 , while 구문 


3) 기타 구문 (= 보조 제어문) :
	  ① continue : 반복 구문에서 이후 실행 구문을 빠져 나가서 반복문의 처음으로 다시 이동 
              ② break : 프로그램 루틴 중간에 강제로 빠져 나가기


2. if 구문 

1) 단순 if 구문 
 	if(조건식){
		조건식이 참이면 실행되는 영역 
	}


2) 양자 택일 구문  
	: 동전 앞면 / 뒷면 , 짝수와 홀수 

	if(조건식){
		조건식이 참이면 실행되는 영역 
	}else{
		조건식이 거짓이면 실행되는 영역 
	}

3) 다중 택일 구문 
	: 학점 매기기 , 주사위 던지기 


	if(조건식1){
		'조건식1'이 참이면 실행되는 영역 
	}else if(조건식2){
		'조건식2'이 참이면 실행되는 영역
	... 
	}else{
		상기 위의 어느 조건도 참이 아니면 실행되는 영역(other)
	} 
	}

3. switch case 구문
  : 중간에 break가 없으면 하단에 모든 코드가 실행이 된다.
  : case 옆에 식은 중복 불가하다.

*수식 표현시 [ ] 는 optional
*expr은 '식'으로써 정수나 문자열이 가능하다. 
	
	switch(expr)}{
	case 식1 : 
		할일 명시
		[break ;]
	case 식2 : 
		할일 명시
		[break ;]
	...
	[default :
		할일 명시]
		[break ;] 	
	}

4. if와 switch의 큰 차이점 

if	switch
조건식    값(정수 , 문자열)

일반적으로 유한개이면 switch가 더 좋다.
