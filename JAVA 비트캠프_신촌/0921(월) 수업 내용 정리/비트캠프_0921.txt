  - 정수 : int
  - 실수 : float , double
  - 문자 : String , char

*변수 : 자료를 읽기 or 쓰기를 수행하기 위한 데이터 저장 공간 
         (ram 에 저장공간 메모리 지정) 

*변수의 정의 or 선언 
:  자료형 변수이름 ;
: 변수의 정의는 무조건 1번만 가능하다. 
: 처음 정의한 변수는 자료형을 바꿔서 다시 선언해도  불가능하다. 

*변수에 값 대입 /할당/쓰기
: 변수이름 = 값;




*식별자 : 변수, 클래스 , 패키지 , 프로젝트 , 메소드 등에 붙이는 이름

*식별자 명명 규칙 
1. 첫 글자는 한글 , 영문 대소문자 , '_' , '$' 로 시작해야한다. 
2. 글자 수에 제한이 없다.
3. 공백 문자나 특수 문자 사용 불가하다. 
4. 숫자는 1번째 아니면 문제 없다. 
5. 대소문자를 구분한다. 
6. 키워드는 사용 불가하다. (키워드 = 자료형) 


*식별자 선언 예시 
String name; // 홍길동 
char ch ; // 문자 'a' 를 저장 하려면  
int age ; // ~의 나이 
boolean bool ; // 진위 값 (참/거짓)

double pi ; // 수학의 파이
