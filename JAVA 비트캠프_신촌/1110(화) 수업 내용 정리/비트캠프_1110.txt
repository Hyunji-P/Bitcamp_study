날짜 함수

데이터베이스에 날짜는 항상 똑같은 값이 들어가 있음 

엑셀에서 사용자서식과 비슷함

똑같은 데이터 가지고 다양한 방식으로 보여주는데
날짜 서식을 바꿔주는 키워드가 'nls_date_format'임

db 작성했다가 빠져나가면 session 이라고 하고

나갔다 들어오면 날짜는 현위치로 바뀜 

ex) 데이터 베이스에서 날짜는 숫자임

1970/1/1 0
1970/1/2 1
1970/1/6 5


저녁 11시 - 오전 11시 = 0.5 (하루절반)

하루는 1.0이고 , 반나절은 0.5이다.

연산의 종류
날짜 + 날짜는 의미가 없음

데이터 베이스의 모든 문자는 외따옴표 
쌍따옴표
	1. 알리아스 이름에 띄어쓰기 존재
	2. 날짜 함수에서 날짜 서식 지정 시
